## Toaster and ToastProvider only props {{ "data-nav": "link", "id": "toaster-provider-props" }}

Can be applied to `ToastProvider` and `Toaster`.

- **limit** - The maximum number of toasts to display at once. Excess toasts are queued and rendered when space becomes available. For no limit, set limit to `false` or `0`. {{ "class" : "mb-2",  "data-nav": "sublink", "id": "limit" }}

     <InlineCode lang="tsx">

  ```tsx
  limit: number | false;
  ```

     </InlineCode>

  Default value: `8` {{ "class": "mt-3 text-xs" }}

- **reverseToastOrder** - If true, toasts are rendered in reverse order (newest at the top/bottom). {{ "class" : "mb-2", "data-nav": "sublink", "id": "reverseToastOrder" }}

    <InlineCode lang="tsx">

        ```tsx
        reverseToastOrder: boolean;
        ```

    </InlineCode>

  Default value: `false` {{ "class": "mt-3 text-xs" }}

- **offsetX** - Horizontal offset for the toaster's position. {{ "class" : "mb-2", "data-nav": "sublink", "id": "offsetX" }}

        <InlineCode lang="tsx">

            ```tsx
            offsetX: number;
            ```

        </InlineCode>

  Default value: `16` {{ "class": "mt-2 text-xs"}}

- **offsetY** - Vertical offset for the toaster's position. {{ "class" : "mb-2", "data-nav": "sublink", "id": "offsetY" }}

        <InlineCode lang="tsx">

            ```tsx
            offsetY: number;
            ```

        </InlineCode>

        Default value: `16` {{ "class": "mt-3 text-xs" }}

- **gutter** - Spacing between toasts. {{ "class" : "mb-2", "data-nav": "sublink", "id": "gutter" }}

        <InlineCode lang="tsx">

            ```tsx
            gutter: number;
            ```

        </InlineCode>

          Default value: `16` {{ "class": "mt-3 text-xs" }}

- **positionY** - Vertical positioning of the toaster. {{ "class" : "mb-2", "data-nav": "sublink", "id": "positionY" }}

        <InlineCode lang="tsx">

            ```tsx
            positionY: "top" | "bottom";
            ```

        </InlineCode>

        Default value: `"top"` {{ "class": "mt-3 text-xs" }}

- **positionX** - Horizontal positioning of the toaster. {{ "class" : "mb-2", "data-nav": "sublink", "id": "positionX" }}

        <InlineCode lang="tsx">

            ```tsx
            positionX: "left" | "center" | "right";
            ```

        </InlineCode>

          Default value: `"right"` {{ "class": "mt-3 text-xs" }}

- **renderOnWindowInactive** - If true, toasts will render even when the window is inactive (blurred). {{ "class" : "mb-2", "data-nav": "sublink", "id": "renderOnWindowInactive" }}

        <InlineCode lang="tsx">

            ```tsx
            renderOnWindowInactive: boolean;
            ```

        </InlineCode>

        Default value: `false` {{ "class": "mt-3 text-xs" }}

- **pauseOnWindowInactive** - If true, toast progress will pause when the window is inactive (blurred). {{ "class" : "mb-2", "data-nav": "sublink", "id": "pauseOnWindowInactive" }}

        <InlineCode lang="tsx">

            ```tsx
            pauseOnWindowInactive: boolean;
            ```

        </InlineCode>

           Default value: `true` {{ "class": "mt-3 text-xs" }}

- **toasterStyle** - Custom styles for the toaster container. {{ "class" : "mb-2", "data-nav": "sublink", "id": "toasterStyle" }}

        <InlineCode lang="tsx">

            ```tsx
            toasterStyle: Omit<JSX.CSSProperties, "top" | "bottom" | "left" | "right">;
            ```

        </InlineCode>

        Default value: `null` {{ "class": "mt-3 text-xs" }}

## Shared props {{ "data-nav": "link", "id": "shared-props" }}

Can be applied to all components; `ToastProvider`, `Toaster` and `Toast`.

- **theme** - The theme of the toast. Two theme types are included; `light` and `dark`, but any string can be provided. {{ "class": "mb-2", "data-nav": "sublink", "id": "theme" }}

    <InlineCode lang="tsx">

  ```tsx
  theme: string | undefined | null;
  ```

    </InlineCode>

      Default value: `"light"` {{ "class": "mt-3 text-xs" }}

- **type** - The type of the toast. Can be "default", "success", "error", "info", "warning" and "loading". {{ "class": "mb-2", "data-nav": "sublink", "id": "type" }}

    <InlineCode lang="tsx">

  ```tsx
  type: "default" | "success" | "error" | "loading" | "warning" | "info";
  ```

    </InlineCode>

  Default value: `"default"` {{ "class": "mt-3 text-xs" }}

- **duration** - Duration (in milliseconds) before the toast is dismissed. Set to `false` or `0` for persistent toasts. {{ "class": "mb-2", "data-nav": "sublink", "id": "duration" }}

    <InlineCode lang="tsx">

  ```tsx
  duration: number | false;
  ```

    </InlineCode>

  Default value: `5000` {{ "class": "mt-3 text-xs" }}

- **onEnter** - CSS animation name when the toast enters. {{ "class": "mb-2", "data-nav": "sublink", "id": "onEnter" }}

    <InlineCode lang="tsx">

  ```tsx
  onEnter: string;
  ```

    </InlineCode>

  Default value: `undefined` {{ "class": "mt-3 text-xs" }}

- **enterDuration** - Duration of the enter animation in milliseconds. {{ "class": "mb-2", "data-nav": "sublink", "id": "enterDuration" }}

    <InlineCode lang="tsx">

  ```tsx
  enterDuration: number;
  ```

    </InlineCode>

  Default value: `300` {{ "class": "mt-3 text-xs" }}

- **onExit** - CSS animation name when the toast exits. {{ "class": "mb-2", "data-nav": "sublink", "id": "onExit" }}

    <InlineCode lang="tsx">

  ```tsx
  onExit: string;
  ```

    </InlineCode>

  Default value: `undefined` {{ "class": "mt-3 text-xs" }}

- **exitDuration** - Duration of the exit animation in milliseconds. {{ "class": "mb-2", "data-nav": "sublink", "id": "exitDuration" }}

    <InlineCode lang="tsx">

  ```tsx
  exitDuration: number;
  ```

    </InlineCode>

  Default value: `300` {{ "class": "mt-3 text-xs" }}

- **onIdle** - CSS class which will be applied when the toast is idle. {{ "class": "mb-2", "data-nav": "sublink", "id": "onIdle" }}

    <InlineCode lang="tsx">

  ```tsx
  onIdle: string;
  ```

    </InlineCode>

  Default value: `undefined` {{ "class": "mt-3 text-xs" }}

- **class** - CSS class applied to the toast body. Can be a string or a function returning a string. {{ "class": "mb-2", "data-nav": "sublink", "id": "class" }}

     <InlineCode lang="tsx">

  ```tsx
  class: string |
    ((args: {
      theme: string | undefined | null;
      type: "default" | "success" | "error" | "loading" | "warning" | "info";
    }) => string);
  ```

     </InlineCode>

       Default value: `"sn-toast"` {{ "class": "mt-3 text-xs" }}

- **style** - Custom styles for the toast body. Can be an object or a function returning an object. {{ "class": "mb-2", "data-nav": "sublink", "id": "style" }}

    <InlineCode lang="tsx">

  ```tsx
  style: JSX.CSSProperties |
    ((args: {
      theme: string | undefined | null;
      type: "default" | "success" | "error" | "loading" | "warning" | "info";
    }) => JSX.CSSProperties);
  ```

    </InlineCode>

       Default value: `undefined` {{ "class": "mt-3 text-xs" }}

- **pauseOnHover** - Whether the toast pauses when hovered. {{ "class": "mb-2", "data-nav": "sublink", "id": "pauseOnHover" }}

    <InlineCode lang="tsx">

  ```tsx
  pauseOnHover: boolean;
  ```

    </InlineCode>

  Default value: `true` {{ "class": "mt-3 text-xs" }}

- **wrapperClass** - CSS class for the toast wrapper. {{ "class": "mb-2", "data-nav": "sublink", "id": "wrapperClass" }}

    <InlineCode lang="tsx">

  ```tsx
  wrapperClass: string;
  ```

    </InlineCode>

  Default value: `"sn-toast-wrapper"` {{ "class": "mt-3 text-xs" }}

- **wrapperStyle** - Custom styles for the toast wrapper. {{ "class": "mb-2", "data-nav": "sublink", "id": "wrapperStyle" }}

    <InlineCode lang="tsx">

  ```tsx
  wrapperStyle: JSX.CSSProperties;
  ```

    </InlineCode>

  Default value: `undefined` {{ "class": "mt-3 text-xs" }}

- **enterCallback** - Callback function executed when the toast enters. {{ "class": "mb-2", "data-nav": "sublink", "id": "enterCallback" }}

    <InlineCode lang="tsx">

      ```tsx
      enterCallback: () => void;
      ```

    </InlineCode>

  Default value: `null` {{ "class": "mt-3 text-xs" }}

- **updateCallback** - Callback function executed when the toast updates. {{ "class": "mb-2", "data-nav": "sublink", "id": "updateCallback" }}

    <InlineCode lang="tsx">

      ```tsx
      updateCallback: () => void;
      ```

    </InlineCode>

  Default value: `null` {{ "class": "mt-3 text-xs" }}

- **exitCallback** - Callback function executed when the toast exits. {{ "class": "mb-2", "data-nav": "sublink", "id": "exitCallback" }}

    <InlineCode lang="tsx">

      ```tsx
      exitCallback: (reason?: boolean | string) => void;
      ```

    </InlineCode>

  Default value: `null` {{ "class": "mt-3 text-xs" }}

- **showDismissButton** - Whether to show a dismiss button. {{ "class": "mb-2", "data-nav": "sublink", "id": "showDismissButton" }}

    <InlineCode lang="tsx">

      ```tsx
      showDismissButton: boolean;
      ```

    </InlineCode>

  Default value: `true` {{ "class": "mt-3 text-xs" }}

- **dismissButtonClass** - CSS class for the dismiss button. {{ "class": "mb-2", "data-nav": "sublink", "id": "dismissButtonClass" }}

      <InlineCode lang="tsx">

      ```tsx
      dismissButtonClass: string;
      ```

      </InlineCode>

  Default value: `"sn-dismiss-button"` {{ "class": "mt-3 text-xs" }}

- **dismissButtonStyle** - Custom styles for the dismiss button. {{ "class": "mb-2", "data-nav": "sublink", "id": "dismissButtonStyle" }}

      <InlineCode lang="tsx">

      ```tsx
      dismissButtonStyle: JSX.CSSProperties;
      ```

      </InlineCode>

  Default value: `undefined` {{ "class": "mt-3 text-xs" }}

- **dismissOnClick** - Whether clicking the toast dismisses it. {{ "class": "mb-2", "data-nav": "sublink", "id": "dismissOnClick" }}

      <InlineCode lang="tsx">

      ```tsx
      dismissOnClick: boolean;
      ```

      </InlineCode>

  Default value: `false` {{ "class": "mt-3 text-xs" }}

- **showProgressBar** - Whether to show a progress bar on the toast. {{ "class": "mb-2", "data-nav": "sublink", "id": "showProgressBar" }}

      <InlineCode lang="tsx">

      ```tsx
      showProgressBar: boolean;
      ```

      </InlineCode>

  Default value: `true` {{ "class": "mt-3 text-xs" }}

- **progressBarClass** - CSS class for the progress bar. {{ "class": "mb-2", "data-nav": "sublink", "id": "progressBarClass" }}

          <InlineCode lang="tsx">

      ```tsx
      progressBarClass: string;
      ```

          </InlineCode>

  Default value: `"sn-progress-bar"` {{ "class": "mt-3 text-xs" }}

- **progressBarStyle** - Custom styles for the progress bar. {{ "class": "mb-2", "data-nav": "sublink", "id": "progressBarStyle" }}

      <InlineCode lang="tsx">

      ```tsx
      progressBarStyle: JSX.CSSProperties;
      ```

      </InlineCode>

  Default value: `undefined` {{ "class": "mt-3 text-xs" }}

- **showIcon** - Whether to show an icon in the toast. {{ "class": "mb-2", "data-nav": "sublink", "id": "showIcon" }}

        <InlineCode lang="tsx">

      ```tsx
      showIcon: boolean;
      ```

        </InlineCode>

  Default value: `true` {{ "class": "mt-3 text-xs" }}

- **icon** - Custom icon for the toast. Can be a JSX element or a function returning a JSX element. {{ "class": "mb-2", "data-nav": "sublink", "id": "icon" }}

        <InlineCode lang="tsx">

      ```tsx
      icon: ((args: args: {
        theme: string | undefined | null;
        type: "default" | "success" | "error" | "loading" | "warning" | "info";
        }) => JSX.Element) | JSX.Element | null;
      ```

        </InlineCode>

  Default value: `null` {{ "class": "mt-3 text-xs" }}

- **dragToDismiss** - Whether dragging the toast dismisses it. {{ "class": "mb-2", "data-nav": "sublink", "id": "dragToDismiss" }}

      <InlineCode lang="tsx">

      ```tsx
      dragToDismiss: boolean;
      ```

      </InlineCode>

  Default value: `true` {{ "class": "mt-3 text-xs" }}

- **dragTreshold** - Distance (in pixels) required to dismiss the toast via dragging. {{ "class": "mb-2", "data-nav": "sublink", "id": "dragTreshold" }}

        <InlineCode lang="tsx">

      ```tsx
      dragTreshold: number;
      ```

        </InlineCode>

  Default value: `100` {{ "class": "mt-3 text-xs" }}

- **ariaLive** - ARIA live attribute for accessibility. {{ "class": "mb-2", "data-nav": "sublink", "id": "ariaLive" }}

      <InlineCode lang="tsx">

      ```tsx
      ariaLive: "polite" | "assertive" | "off";
      ```

      </InlineCode>

  Default value: `"polite"` {{ "class": "mt-3 text-xs" }}

- **role** - ARIA role attribute. {{ "class": "mb-2", "data-nav": "sublink", "id": "role" }}

      <InlineCode lang="tsx">

      ```tsx
      role: "status" | "alert" | "log";
      ```

      </InlineCode>

  Default value: `"status"` {{ "class": "mt-3 text-xs" }}

## Special props: {{ "data-nav": "link", "id": "special-props" }}

- **id** - Unique identifier for the toast. If not provided, a random ID is generated. {{ "class": "mb-2", "data-nav": "sublink", "id": "id" }}

    <InlineCode lang="tsx">

      ```tsx
      id: string;
      ```

    </InlineCode>

- **content** - The main content of the toast. Can be a string, JSX element, or a function returning JSX. {{ "class": "mb-2", "data-nav": "sublink", "id": "content" }}

    <InlineCode lang="tsx">

      ```tsx
      content: ToastContent;
      ```

    </InlineCode>

- **contentType** - Determines whether the content is static or dynamic. {{ "class": "mb-2", "data-nav": "sublink", "id": "contentType" }}

    <InlineCode lang="tsx">

      ```tsx
      contentType: "static" | "dynamic";
      ```

    </InlineCode>

  Default value: `"static"` {{ "class": "mt-3 text-xs" }}

- **toasterId** - Specifies the toaster instance where the toast should be displayed. {{ "class": "mb-2", "data-nav": "sublink", "id": "toasterId" }}

    <InlineCode lang="tsx">

      ```tsx
      toasterId: string;
      ```

    </InlineCode>
