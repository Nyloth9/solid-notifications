# Quickstart

Get up and running with SolidNotifications in just a few simple steps. {{ "class": "lead" }}

## Installation {{"data-nav": "link"}}

Install the package from npm: {{ "class": "lead" }}

<Code lang="bash">

```bash
npm install solid-notifications
# or
yarn add solid-notifications
# or
pnpm add solid-notifications
```

</Code>

## Usage {{"data-nav": "link"}}

Here's a quick example to show how easy it is to use: {{ "class": "lead" }}

### 1. Mount the ToastProvider and Toaster components:

Wrap your app with the `ToastProvider` component. Put the `Toaster` component anywhere in the component tree to display toasts.

<Code lang="tsx">

```tsx
import { ToastProvider, Toaster } from "solid-notifications";

export default function App(props) {
  return (
    <ToastProvider>
      <Toaster />
      {props.children}
    </ToastProvider>
  );
}
```

</Code>

### 2. Import the Solid Notifications css file into your main css file:

It can also be imported directly into the main component file.

<Code lang="css">

```css
/* index.css */

@import "solid-notifications/index.css";
```

</Code>

### 3. Create toasts:

<Code lang="tsx">

